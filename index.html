O Sistema Hexadecimal

Enquanto o sistema decimal e seus dez dígitos estão profundamente enraizados em nossas vidas diárias, a ciência da computação e o processamento de dados dependem fortemente do sistema binário, ou código binário. O sistema binário torna possível representar situações complexas com apenas dois estados: 0 e 1. No entanto, grandes números binários rapidamente se tornam confusos. É aqui que o sistema hexadecimal pode ajudar. As informações expressas com oito dígitos no sistema binário podem ser expressas com apenas dois números hexadecimais.

Para que é usado o sistema hexadecimal?

A palavra hexadecimal é composta pelos termos hexa e decimal. Hexa vem do grego e significa “seis”, enquanto decem é a palavra latina para “dez”. O sistema hexadecimal é, portanto, um sistema de valor posicional que representa números usando a base 16. Isso significa que o sistema hexadecimal usa 16 dígitos diferentes. Ou seja, são 16 dígitos possíveis, em contraste com os dois do sistema binário (0 e 1) e os dez do sistema decimal (0 a 9). Mas qual é o propósito do sistema?

Para que é usado o sistema hexadecimal?

O sistema hexadecimal é usado na tecnologia de computadores e torna mais legíveis números grandes e sequências de bits longas. Eles são agrupados em seções de quatro bits e depois convertidos em números hexadecimais. O resultado é que uma longa sequência de uns e zeros é transformada em números hexadecimais mais curtos, que por sua vez podem ser divididos em grupos de dois ou quatro. Os números hexadecimais são, portanto, uma forma mais compacta de representar sequências de bits. O sistema é usado, por exemplo, para endereços de origem e destino em protocolos de Internet (IPs) , em códigos ASCII ou para descrever códigos de cores em web design com a linguagem de folha de estilo CSS.

Sistema hexadecimal: como escrevê-lo?

Conforme mencionado acima, o sistema hexadecimal utiliza 16 dígitos. É aqui que nos deparamos com um problema potencial. Na nossa forma tradicional de escrever números usamos os números decimais 10, 11, 12, 13, 14 e 15, que consistem em dois símbolos reunidos. Portanto, se você usasse o número 10 em notação hexadecimal, não ficaria claro o que você quer dizer.

Para evitar esse problema, as letras de A a F são usadas para representar os números de dez a 15 no sistema hexadecimal. Portanto, no total, os números de 0 a 9 e de A a F são usados para representar os equivalentes de números binários e números decimais. Existem várias notações disponíveis para distinguir números hexadecimais de números decimais:

  • 76 16
  • 73 hexadecimal
  • 73h
  • 73H
  • 0x73
  • R$ 73
  • #73
  • "73
  • X'73'

O prefixo 0x e o sufixo h são especialmente comuns em programação, e o prefixo cifrão $ é freqüentemente usado em certas famílias de processadores em linguagem assembly.
